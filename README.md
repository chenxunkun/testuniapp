# testuniapp
A uni-app project for front-end development, supporting cross-platform operation, simple structure and easy secondary development.

## Project Introduction
testuniapp is a lightweight front-end project developed based on uni-app framework, which can be quickly compiled and run on multiple platforms (App, H5, small programs, etc.). It has a concise project structure and low threshold for secondary development, suitable for rapid prototype construction and small and medium-sized front-end project development.
