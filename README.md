# testuniapp
A uni-app project for front-end development learning and testing, focusing on multi-terminal tourist attraction display. It supports cross-platform operation, has a simple structure and is easy for secondary development, which can quickly adapt to multiple terminals and provide users with a convenient and intuitive experience of browsing and querying tourist attractions.

## Project Introduction
testuniapp is a lightweight front-end project developed based on the uni-app framework, with the core positioning of multi-terminal tourist attraction display. It can be quickly compiled and run on multiple platforms such as App, H5, and mini programs, with a simple structure and low threshold for secondary development. It supports the display of basic scenic spot information and is suitable for the rapid construction of tourism-related multi-terminal projects.
